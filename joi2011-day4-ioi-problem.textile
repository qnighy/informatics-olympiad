20XX 年，ついに JOI 国で行われることになった IOI には <var>K</var> 人の選手が参加した．選手には <var>1, 2, …, K</var> と番号が付けられている．問題は全部で <var>N</var> 問出題され，各選手は各問題について <var>0</var> 以上 <var>100</var> 以下の整数の点数を付けられる．

選手には <var>N</var> 問の合計点に応じてメダルが与えられる．メダルの与えられる詳しい条件は，たとえば金メダルについては次のように決まっている：

* <var>G</var> を，<var>N</var> 問の合計点が <var>G</var> 点以上の選手の人数が全体の <var>\frac{1}{12}</var> 以上となるような最大の値とする．このとき，金メダルが与えられる条件は，<var>N</var> 問の合計点が <var>G</var> 点以上であることである．

すでに競技が終了している問題が <var>M</var> 問あり，点数が確定している．IOI のウェブサイトで各選手の現在までの合計点を見ていたあなたは，金メダルを与えられることが確実な選手や金メダルを与えられる可能性のある選手がそれぞれどれだけいるのかが知りたくなった．

h2. 課題

各選手の現在までの合計点が与えられたとき，金メダルを与えられることが確実な選手，および，金メダルを与えられる可能性のある選手をそれぞれ番号順に出力するプログラムを作成せよ．

h2. 制限

| <var>1 \leq K \leq 100\,000</var> | 選手の人数 |
| <var>1 \leq N \leq 10\,000\,000</var> | 全体の問題数 |
| <var>0 \leq M \leq N</var> | すでに終了した問題数 |
| <var>0 \leq P_i \leq 100 \times M</var> | 番号 <var>i</var> の選手の現在までの合計点 |

h2. 入力

標準入力から以下の入力を読み込め．

* <var>1</var> 行目には整数 <var>K, N, M</var> が空白を区切りとして書かれており，選手の人数が <var>K</var> 人であること，全体の問題数が <var>N</var> 問であること，そのうちすでに終了した問題が <var>M</var> 問あることを表す．
* 続く <var>K</var> 行には各選手の得点が書かれている．<var>i + 1</var> 行目 <var>(1 \leq i \leq K)</var> には，番号 <var>i</var> の選手の現在までの合計点を表す整数 <var>P_i</var> が書かれている．

h2. 出力

標準出力に以下の内容を出力せよ．

* 初めの <var>a</var> 行は，金メダルを与えられることが確実な選手の番号を，<var>1</var> 行に <var>1</var> つずつ，小さい順に列挙していなければならない．ただし <var>a</var> は金メダルを与えられることが確実な選手の人数である．
* 続く <var>1</var> 行には文字列 ==--------== (ハイフンが <var>8</var> 個) を出力せよ．
* 続く <var>b</var> 行は，金メダルを与えられる可能性のある選手の番号を，<var>1</var> 行に <var>1</var> つずつ，小さい順に列挙していなければならない．ただし <var>b</var> は金メダルを与えられる可能性のある選手の人数である．

h2. 採点基準

* 採点用データのうち，配点の <var>20%</var> 分については，<var>K \leq 3\,000</var> を満たす．
